Key Insights (from Memory)
CRITICAL: "AI-generated" String in 100% of AI Content
Every AI-labeled row contains the literal string "AI-generated" inside the content column. A single string check achieves 100% accuracy. This is the most severe form of data leakage possible — the label is written into the content.
Status: Confirmed — dataset-breaking
CRITICAL: Source Column Perfectly Predicts Label
source="generated" maps exclusively to AI (10,004 rows). blog/news/github map exclusively to human (9,996 rows). No detection intelligence required — just a lookup.
Status: Confirmed — dataset-breaking
CRITICAL: ai_model Column Is 100% Predictive
All 9,996 human rows have a blank ai_model. All 10,004 AI rows have it filled (Claude / GPT-4 / Gemini). Three independent columns each achieve 100% label prediction.
Status: Confirmed — dataset-breaking
58.8% of Rows Are Duplicate Content
11,758 rows share duplicate content. Top repeated entries are identical Java/C++ class skeletons repeated 340–360 times each. The dataset is largely synthetic and templated.
Status: Confirmed
AI Code Is More Verbose Than Human Code
In code (structured) content: AI outputs are measurably longer — +2.1 words, +15 chars. In text (unstructured): AI is virtually identical to human. AI does not behave uniformly across content types.
Status: Confirmed
Complexity Score Is Randomly Assigned
Complexity score is uniformly distributed across 1–10 in ALL four groups with no meaningful skew. This column was randomly assigned — it is pure noise and should be excluded from any analysis.
Status: Confirmed
Dataset Is Artificially Balanced
Near-perfect 50/50 split (49.98% human vs 50.02% AI). Real-world AI detection datasets are typically imbalanced. Performance metrics on this data will not reflect real deployment conditions.
Status: Confirmed
Prompt Phrasing Shows No Label Signal
The first verb in prompt phrasing shows no systematic differences between human and AI labels. Same verbs used proportionally for both. This is one of the few clean columns in the dataset.
Status: Confirmed
