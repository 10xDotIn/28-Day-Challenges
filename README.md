# 28-Day Agentic Analysis Challenge

> **Data Analytics is Dead. Agentic Analysis is the Future.**

Welcome to the **28-Day Agentic Analysis Challenge** — a hands-on, project-based learning experience where we prove that one person equipped with the right agentic skills can replace an entire data science team.

---

## What is This Challenge?

Traditional data analytics — the dashboards, the manual wrangling, the endless Tableau/Power BI drag-and-drop workflows — is being disrupted. The era of **Agentic Analysis** has arrived, where AI agents handle the heavy lifting, and a single skilled operator drives insights faster than a team of 10 ever could.

In this **28-Day Challenge**, we go LIVE every single day, build a real-world project from scratch, and push the completed work here so you can follow along, practice, and build your own portfolio.

---

## Who is This For?

- Data Analysts tired of repetitive dashboard building
- Data Scientists looking to 10x their productivity
- Business Analysts who want to stay ahead of the curve
- Students & Career Switchers entering the data field in 2026
- Anyone who wants to learn how **one person can do the work of an entire data team**

---

## What You'll Learn

| Traditional (Old Way) | Agentic (New Way) |
|---|---|
| Manual data cleaning in Excel/Python | AI agents that clean, transform & validate data autonomously |
| Drag-and-drop Tableau/Power BI dashboards | Agentic dashboards that build themselves from natural language |
| Writing SQL queries from scratch | Agents that understand your data and generate optimized queries |
| Static reports & presentations | Dynamic, self-updating analysis pipelines |
| Teams of 5-10 analysts per project | **1 person** running the entire workflow |

## How to Use This Repo

1. **Star** this repo to stay updated
2. **Watch** for daily pushes after each live session
3. **Clone** the repo to follow along locally:
   ```bash
   git clone https://github.com/10xDotIn/28-Day-Challenges.git
   ```
4. Navigate to the day's folder to access datasets, code, and project files
5. Try building the project yourself before watching the live session
